2. Estruturas Condicionais (if/else)

O `if` e `else` permitem que o programa tome decisões. Este exemplo verifica se você tem idade para entrar em uma montanha-russa.

Código de Exemplo:

#include <iostream> using namespace std; int main() { int idade; cout << "Qual a sua idade? "; cin >> idade; if (idade >= 12) { cout << "Pode entrar! Divirta-se!" << endl; } else { cout << "Que pena! Voce ainda nao pode entrar." << endl; } return 0; }

3. Laços de Repetição

Laços repetem blocos de código. O `for` é ótimo para um número fixo de repetições, enquanto o `while` continua até uma condição se tornar falsa.

Laço `for`

for (int i = 0; i < 3; i++) { cout << "GOL!" << endl; }

Laço `while` (Jogo de Adivinhação)

int numeroSecreto = 7; int palpite = 0; while (palpite != numeroSecreto) { cout << "Qual o seu palpite? "; cin >> palpite; } cout << "Parabens! Voce acertou!" << endl;
